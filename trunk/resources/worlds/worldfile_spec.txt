//World files have the extension .wld and specify a set of levels with a particular theme,
//they are laid out as follows:


// The theme name is defined first, this will be the theme for all
// levels loaded in this world
<theme_name> 

// The world name - as it appears to people in the splash screen before playing
<world_name>

// The picture associated with the world, used when selecting the world in the level/world select menu
<image_filepath>

// The level list comes next, this is a list of all the level files, in order,
// for this world - no extensions are included
<level_1_name>
<level_2_name>
//...
<level_n_name>
